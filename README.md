[JavaScript Fighting Game Tutorial with HTML Canvas](https://www.youtube.com/watch?v=vyqbNFMDRGQ)

## Commands

### Player 1: Samurai Mack

- Move Right: `d`
- Move Left: `a`
- Jump: `w`
- Attack: `space`

### Player 2: Kenji

- Move Right: `right-arrow`
- Move Left: `right-arrow`
- Jump: `up-arrow`
- Attack: `down-arrow`
